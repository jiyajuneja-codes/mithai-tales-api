# Mithai Tales API 🪔
A REST API built with Express.js as part of an academic assignment.
The project is inspired by my frontend website "Mithai Tales", which explores the history of Indian Sweets.

# Project Context
The frontend website was created earlier and this backend API is built to [practice and demonstrate core Express.js concepts such as 
routing, middleware, route parameters, query parameters and request bodies.

## Frontend (for reference): https://mithaitales.netlify.app/
# Built with:
. Node.js
. Express.js
. Morgan (third-party middleware)

# Routes
. GET /- Welcome message
. GET /sweets/:name - Route parameters
. POST /sweets/add - Request body
. GET /sweets?era=mughal - Query parameters
