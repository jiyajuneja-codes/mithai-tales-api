# Mithai Tales API 🪔
A simple REST API built using Express.js as part of an academic assignment. This project is inspired by my frontend website “Mithai Tales”, which explores the history and cultural background of Indian sweets.

The purpose of this API is to practice and demonstrate core backend concepts taught during the course.

# Frontend Reference: https://mithaitales.netlify.app/⁠

# Built With
- Node.js
- Express.js
- Morgan (third-party middleware)
- Custom middleware (logging request method, URL, time, and IP)

# API Routes
- GET / — Welcome message
- GET /sweets/:name — Demonstrates route parameters
- GET /sweets?era=mughal — Demonstrates query parameters
- POST /sweets/add — Demonstrates request body handling

# Future Improvements
This project currently focuses on Express.js fundamentals. As I continue learning backend development, I plan to:
- Integrate MongoDB to store sweet history data dynamically
- Add CRUD operations instead of static data
- Implement search and filter features
- Connect this API with the frontend website
- These improvements will help convert this into a more complete full-stack project.
